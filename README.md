# xcperfect

xcperfect makes xccov outputs prettier

## Installation

```bash
$ gem install xcperfect
```

## Usage

```bash
$ xcrun xccov --json [options] | xcperfect
```

or 

```bash
$ xcrun xccov --json [options] > report.json
$ xcperfect report.json
```

## Formats

- `--all`, `-a` **Default**

- `--simple`, `-s`
