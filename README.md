## Installation

```bash
$ gem install xcperfect
```

## Usage

- Pipe

```bash
$ xcrun xccov --json [options] | xcperfect [options]
```

- Feed it in via flags

```bash
$ xcperfect [--file | -f] report.json [options]
```

> ⚠️ Please note that `xcperfect` only takes in `xccov` json output

## Formats

- `--all`, `-a` **Default**

- `--simple`, `-s`
